# Pizza-Sales--Analysis-project

# Project Overview

This project focuses on analyzing pizza sales data to uncover valuable business insights using MySQL Workbench for data analysis and Excel for visualization and dashboard creation.

The goal of this project is to transform raw sales data into actionable insights that help improve business performance and decision-making.

# Tools & Technologies

* SQL (MySQL Workbench)
* Microsoft Excel (Dashboard & Visualization)
  
# Key Business Metrics Analyzed

* Total Revenue
* Average Order Value
* Total Orders
* Total Pizzas Sold
* Average Pizzas per Order
  
# SQL Analysis Performed

#Based on your SQL file, you did:

* Revenue calculation using SUM(total_price)
* Order-level analysis using COUNT(DISTINCT order_id)
* Time-based insights:
    * Daily trends (DAYNAME)
    * Hourly trends (HOUR)
* Category & size contribution analysis:
    * % sales by category
    * % sales by size
* Product performance:
    * Top 5 best-selling pizzas
    * Bottom 5 least-selling pizzas
      
# Dashboard Features (Excel)

* Sales trends visualization
* KPI summary cards
* Category & size breakdown charts
* Peak sales time identification
* Product performance insights

# Key Insights

* Identified peak business hours and days
* Found highest revenue-generating pizza categories
* Highlighted top-performing and underperforming products
* Analyzed customer ordering patterns
